# IFS-TRANSLATOR
Este proyecto tiene como proposito el tratar de minimizar el espacio de error que puede existir al momento de crear comentarios de reparacion basados en el travel del proceso.
El proyecto es un formulario sencillo el cual solicita al usuario campos especificos del travel para formar lo que se conoce como comentarios de reparacion.
Cuando se toman todos los campos y se sube el formulario este traduce cada campo y lo retorna en un formato especifico para que el usuario tome la salida.

# Nota
Este proyecto utiliza una API de traduccion externa llamada Deep Translate para realizar la traduccion de todos los campos del formulario. (Se utiliza una API KEY de este servicio)
[Deep Translate API](https://rapidapi.com/gatzuma/api/deep-translate1)

```bash
   npm install -g pnpm
```

```bash
   cd next-mb
   pnpm install

```
